Features

必要なものだけ、過不足なく

派手な機能より、毎日開いて使えることを優先した。

月カレンダー

課題・試験・授業を8種類のカテゴリで色分け。繰り返し登録・複数日登録に対応。

ダッシュボード

今月の予定数・近日のイベント一覧をひと目で確認。科目・種別でフィルタリング可能。

時間割

7日×6〜7時限のグリッド。科目とアイコンをセルごとに設定してクラウド保存。

クラウド同期

Supabaseによるリアルタイム同期。

Stack

シンプルな技術構成

フレームワークなし。バンドルサイズを最小に保ちながら、必要な機能だけを実装している。

Vanilla JS HTML / CSS Supabase Netlify Open Source esbuild
